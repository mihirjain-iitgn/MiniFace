Depedecies
	Server :

		python 3.x

	Client :

		python 3.x

		stdiomask : It can be installed with pip3 install stdiomask or pip install stdiomask depending on how the path varibles in the ystem.


Note :
	For Running multiple client from in a single machine, copies of the client file must be made and each file must be put in a different directory.
	This is because client maintains a file for storing session id and running multiple clients will cause collisions in the data of that file.


Execution :
	run python server.py or python3 server.py in one terminal.
	run python client.py or python3 client.py in another terminal.
